2. General Information

The NOW app can be used without registration.

No personal data such as name, email, or phone number is collected.

All stored data is non-personal and serves exclusively for the app's functionality.

3. What Data We Store

a) Nickname

When first starting the app, users choose a freely invented nickname.

This nickname is used exclusively for display within the app and is not personal data.

b) Location (freely entered)

Users can voluntarily enter an approximate location (e.g., city, country, or any freely chosen term).

This information is not precise and does not allow identification of a person.

c) Ideas / Micro-Action Suggestions & Push Notifications

When users submit an idea, we store:

  • Nickname
  • Voluntary location
  • Idea text
  • Device push notification token (only if you explicitly opt-in during submission)

If you opt-in for notifications, your device token is stored securely and used solely to notify you when your idea gets scheduled. No other messages are sent.

You can opt-out at any time in your profile settings.

d) Slot Data

For the micro-action slots to function, we store:

  • Which slot was used
  • Which micro action was displayed
  • Nickname and location for display on share cards or community notices

4. Firebase Firestore (Data Storage)

We use Firebase Firestore to store the above-mentioned data.

This data is not linked to personal information.

5. Firebase Analytics (Anonymous Usage Statistics)

NOW uses Firebase Analytics to collect anonymous usage data.

Firebase Analytics does not collect personal data, does not use advertising IDs, and does not create user profiles.

6. Push Notifications (Firebase Cloud Messaging)

If you opt-in when submitting an idea, NOW uses Firebase Cloud Messaging (FCM) to notify you when your idea is scheduled.

  • Notifications are only sent to users who explicitly opted-in.
  • We do not send marketing or unsolicited push notifications.
  • Your device token is not shared with any third party outside of Google/Firebase.
  • You can withdraw consent at any time in your profile settings or by contacting us.

7. No Sharing with Third Parties

No stored data is shared with third parties outside of Firebase/Google.

We do not operate advertising, tracking, or sell any data.

8. No Location Tracking / No Sensitive Data

NOW does not collect GPS location data, does not use tracking tools, does not integrate social media SDKs, and does not use cookies.

9. User Rights

Users can request deletion of the following data at any time:

  • Nickname
  • Location
  • Submitted ideas
  • Slot data
  • Push notification token

Contact: sandrazaenglein@gmail.com

We will delete the data within 14 days.

10. Data Retention

Data remains stored as long as necessary for the app's functionality.

No profiling or identity tracking takes place.
