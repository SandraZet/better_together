2. General Information

The NOW app can be used without registration.

No personal data such as name, email, or phone number is collected.

All stored data is non-personal and serves exclusively for the app's functionality.

3. What Data We Store

a) Nickname

When first starting the app, users choose a freely invented nickname.

This nickname is used exclusively for display within the app and is not personal data.

b) Location (freely entered)

Users can voluntarily enter an approximate location (e.g., city, country, or any freely chosen term).

This information is not precise and does not allow identification of a person.

c) Ideas / Micro-Action Suggestions

When users submit an idea, we store:

  • Nickname
  • Voluntary location
  • Idea text

This data is used to review, display, or editorially process the idea within the app.

d) Slot Data

For the micro-action slots to function, we store:

  • Which slot was used
  • Which micro action was displayed
  • Nickname and location for display on share cards or community notices

4. Firebase Firestore (Data Storage)

We use Firebase Firestore to store the above-mentioned data.

This data is not linked to personal information.

5. Firebase Analytics (Anonymous Usage Statistics)

NOW uses Firebase Analytics to collect anonymous usage data, e.g.:

  • App opened
  • Slot completed
  • Certain screens accessed

This data helps us improve the app.

Firebase Analytics:

  • Does not collect personal data
  • Does not use advertising IDs (IDFA/AAID)
  • Does not create user profiles
  • Does not perform cross-app tracking
  • Serves exclusively for anonymous usage analysis

An opt-out option in the app is currently not provided, as no personal or advertising-related data is collected.

6. No Sharing with Third Parties

No stored data is shared with third parties outside of Firebase/Google.

We do not operate advertising, tracking, or sell any data.

7. No Location Tracking / No Sensitive Data

NOW:

  • Does not collect GPS location data
  • Does not use tracking tools
  • Does not integrate social media SDKs
  • Does not use cookies

The entered location is a freely entered text and not an actual location.

8. User Rights

Users can request deletion of the following data at any time:

  • Nickname
  • Location
  • Submitted ideas
  • Slot data

Contact: sandra@szetstudios.com

We will delete the data within 14 days.

9. Data Retention

Data remains stored as long as necessary for the app's functionality.

No profiling or identity tracking takes place.

10. Changes to Privacy Policy
